**********Runtime environment**********
This project is run on Microsoft Visual Studio 2022 on windows10 system.

Language:C++


**********How to use mica_profile**********
1,Open "mica_profile.vcxproj". Open file "main.cpp".

2,Set the configurations below in "main.cpp":
#define FILE_IN  "xxx"  .It means input file name. This file is in the same folder as project.
#define PROJRCT_PATH "xxx"	.It means project path. 
#define SIZE xxx     .It is interval size
#define NAME "xxx"    .It means output file name: NAME.txt
#define ISAARCH64 1/0  .      Set ISAARCH64 to 1 to analyse 64bit trace. Set ISAARCH64 to 0 to analyse 32bit trace. 

An example configuration:
#define FILE_IN "instrace.new.log"
#define PROJRCT_PATH "E:\\VS2022\\projects\\mica_profile"
#define SIZE 1000000												
#define NAME "instrace.test"											
#define ISAARCH64 1	

3,Run main.cpp. Output files will be generated at folder "Trace_Summaryfea" and folder "Trace_Summarymica" respectively.

**********Debug several instructions**********
In main.cpp, make enable #define TEST_SINGLE_INST	     to start testing several instructions. The messages of instructions are input by modifying the struct "InstTrace" between line 199 and line 203.
